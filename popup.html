This extension uses OpenAI's API to enhance your LinkedIn experience by automatically summarizing posts, extracting key points, answering questions, and providing translation capabilities.

Features

  • Automatic post summarization
  • Ask questions about any post
  • Key Points extraction
  • Multilingual support with RTL detection
  • Translation between English and Persian
